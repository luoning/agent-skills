# Phase 5: Semantic HTML5 Layout

## Trigger
Use when anchor mapping (Phase 4) is ready, and you are starting HTML layout construction.

---

## 1. Structural Slot Rules
Construct pure layouts using container classes in `layout.css`:
*   `.container`: Structural centering (max-width `1200px`).
*   `.grid-3` / `.grid-2`: Box layout structures.
*   **No styling elements**: `background`, `border`, `color` are strictly prohibited in the layout structures.

---

## 2. Definition of Done (DoD)
*   [ ] The page HTML markup is created using semantic elements (`<main>`, `<section>`).
*   [ ] The `.pipeline_state.json` phase 5 status is marked as `completed`.
