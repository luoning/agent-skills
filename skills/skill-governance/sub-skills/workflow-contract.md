# Sub-skill: Workflow Contract (Ideation Sandbox)

## Overview
Governs the collaborative pipeline between the agent and the user, isolating unapproved brainstorms.

## 1. Rules
- **Sparks (畅想)**: Keep raw brainstorming strictly in temporary scratch directories.
- **Opinions (观点)**: Evolving concepts requiring reuse are mapped locally.
- **Solutions (方案)**: Production-grade skills/code must be approved before save commands.
- **Isolation**: No files may be staged or committed inside the active workspace until the user explicitly issues "save".
