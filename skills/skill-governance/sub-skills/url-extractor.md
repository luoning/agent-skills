# Sub-skill: URL Extractor (POE Value Miner)

## Overview
Retrieves external web content, deduces sharing intent, and processes data structures without sycophancy or buzzwords.

## 1. Rules
- **Intent Deduction**: Guess the user's intent by mapping the content to active workspace projects.
- **POE Triad Mapping**:
  - **People (人)**: Actors and user frictions.
  - **Objects (物)**: System components, layout parameters, and assets.
  - **Events (事)**: Transitions, loops, and critical transaction flows.
- **Tone**: Speak in plain-spoken Chinese. Reject marketing terms ("empower", "closed-loop").
